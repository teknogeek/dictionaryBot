dictionaryBot
=============

A Simple IRC Dictionary Bot written in java

=============

Usage: 
 
!define [word(s)]
    -Uses Wordnet and falls back to Urban Dictionary

!defineud [word(s)]
    -Forces Urban Dictionary 
    


Links:

Wordnet: http://wordnet.princeton.edu/
Urban Dictionary: http://urbandictionary.com
